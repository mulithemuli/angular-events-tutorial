Lesson two - choosing colors

There exists a dynamic list of colored boxes (<app-color-box>). The color of each box can be set individually via the colorpicker inside. Changing its color changes the color of the colored box. Besides the colorpicker there is a checkbox called master.

The task is to implement a feature when the master checkbox is checked the color of all the other boxes changes with the change of the master box.
When there are multiple masters the last change applies to all the color boxes.
