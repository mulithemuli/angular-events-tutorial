Lesson one - adding pets

This lesson covers two tasks.

The first task is to enter some data into the form provided in <app-pet-form>. The data entered there should be added to the list of animals inside this component, <app-pet-collection>.
What makes this interaction possible?

The second task is to select an animal in the table which will then show up inside the form below. Here the properties can be changed. The second button should reset the form and a new animal can be added without modifying existing ones. This one is a bit tricky and you should consider using events instead of just setting values.

The Bonus task is to sort the animal list. Even for new animals.
